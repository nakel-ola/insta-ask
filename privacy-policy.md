# InstaAsk Privacy Policy

**Last Updated:** March 2026

InstaAsk is a Chrome extension that allows users to highlight text on a webpage and receive short AI-generated answers. This Privacy Policy explains what information the extension processes and how it is used.

---

## 1. Information We Process

InstaAsk processes only the information necessary to provide its functionality.

### 1.1 Selected Website Content
When a user highlights text on a webpage and clicks the **Ask** button, the selected text may be sent to an AI service provider to generate an answer.

This text is only sent when the user explicitly interacts with the extension.

### 1.2 Page URL
The extension stores the **URL of the page** where a question was asked so that previously answered questions can be grouped and displayed by website inside the extension popup.

### 1.3 API Key
If the user provides an **OpenAI API key**, it is stored locally in the user's browser using Chrome's local storage.

The API key is used only to send requests to the AI service provider.

The API key is **never transmitted to the developer**.

---

## 2. How Information Is Used

The information processed by InstaAsk is used solely to:

- Generate answers to highlighted text using AI
- Display answers within the extension interface
- Save and display question history grouped by webpage URL

The extension does **not use this data for advertising, tracking, or profiling purposes**.

---

## 3. Data Storage

All stored data is saved locally on the user's device using **Chrome's local storage system**.

This includes:

- The user's API key
- Question and answer history
- Associated webpage URLs

No data is stored on servers controlled by the developer.

---

## 4. Third-Party Services

When a user requests an answer, the selected text may be sent to an external AI service provider (such as **OpenAI**) in order to generate the response.

These services process the text according to their own privacy policies.

InstaAsk does not control how third-party providers handle data once it is sent to them.

---

## 5. Data Sharing

InstaAsk **does not sell, trade, or transfer user data to third parties**.

User data is only sent to external AI services when necessary to generate the requested answer.

---

## 6. User Control

Users have full control over their data.

Users can remove stored data by:

- Clearing the extension's stored data
- Removing the extension from Chrome

---

## 7. Changes to This Privacy Policy

This Privacy Policy may be updated if the extension's functionality changes.

Users are encouraged to review this page periodically for updates.
